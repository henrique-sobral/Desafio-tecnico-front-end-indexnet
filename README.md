# Desafio Indexnet

Interface front-end para configuracao inicial de um site de pet shop, com fluxo em multiplas etapas e tela de sucesso ao final do cadastro.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- Google Fonts (`Poppins`)
- SVG para icones e ilustracoes

## Estrutura do projeto

- `index.html`: pagina principal com o formulario em etapas
- `style.css`: estilos da interface e responsividade
- `script.js`: logica de navegacao entre etapas, validacoes e mascaras
- `success.html`: tela exibida apos a finalizacao do cadastro
- `img/`: arquivos visuais utilizados no projeto

## Referencias

- Layout e proposta visual baseados no Figma de uma referência que encontrei para o projeto
- Tipografia utilizada via Google Fonts
